# fx-991 CN X 简明使用说明

> 本项目是针对全体上海市高中生的一份计算器（fx-991 CN X）使用的简单手册，致力于帮助更多同学更好的压榨手中计算器的功能。


## Quick Start
```bash
$ npm i docsify-cli -g
$ git clone git@github.com:Howardzhangdqs/fx-991CN-X-Usage.git
$ cd fx-991CN-X-Usage
$ docsify serve ./docs
```

?> 请确保您所使用的设备中已配置好可用的`Node.js`、`npm`与`git`

打开<localhost:3000>即可阅读
