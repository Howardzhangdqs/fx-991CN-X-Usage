# 存储器

## 答案存储器#bAns#

#bAns#仅会存储上一次的计算结果（按了#b=#后除了结果才算计算结果）

可直接当成变量使用。

## 变量

计算器内预设了9个变量：$x$、$y$、$\mathrm{A}$、$\mathrm{B}$、$\mathrm{C}$、$\mathrm{D}$、$\mathrm{E}$、$\mathrm{F}$、$\mathrm{M}$

### 变量赋值
